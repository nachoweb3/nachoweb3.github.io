## Anatomía de un NFT scam

### Red flags que debes ver:

**1. Contrato no verificado**

Si no puedes ver el código en Etherscan = **HUYE**

**2. Equipo anónimo**

"Por privacidad no revelaremos identidades" = Scam incoming

**3. Promesas imposibles**

"Guaranteed 10x"
"Passive income forever"
"Partnership con Disney" (siempre mentira)

**4. FOMO artificial**

"SOLO QUEDAN 10 NFTs"
"MINT TERMINA EN 1 HORA"
"BALLENAS COMPRANDO"

Todo fabricado.

**5. Metadata centralizada**

Si la imagen está en servidor web normal (no IPFS) = pueden cambiarla

Tu "Bored Ape" puede convertirse en dick pic.

### Técnicas de estafa:

**Rug Pull Clásico:**
```
1. Crea proyecto hype
2. Vende todo
3. Desaparece
4. Profit
```

**Wash Trading:**
```
1. Vende NFT a ti mismo con otra wallet
2. Creas "volumen" falso
3. Víctima ve actividad y compra
4. Tú desapareces
```

**Phishing Discord:**
```
1. Hackea Discord del proyecto
2. Anuncia "surprise mint"
3. Link falso roba tu wallet
4. Adiós fondos
```
